## ColonyDetectionEvaluation 菌落偵測實驗
* 菌落偵測實驗評估方法
    * True Positive (TP): 原本是菌落，系統判斷為菌落
    * True Negative (TN): 原本不是菌落，系統未將其判斷為菌落
    * False Positive (FP): 原本不是菌落，系統判斷為菌落
    * False Negative (FN): 原本是菌落，系統未將其判斷為菌落
